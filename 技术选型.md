# SmartSupport AI 技术方案选型

## 企业级智能客服 Agent（个人项目版）

---

# 一、选型原则

本项目为 **个人 AI 工程实践项目**，技术选型遵循以下原则：

1. **开源优先**
   所有核心组件优先使用开源方案，避免依赖商业 SaaS 服务。

2. **轻量化优先**
   避免复杂分布式架构，优先选择本地可运行组件。

3. **简单易用优先**
   技术方案应降低开发复杂度，使项目能够快速完成 MVP。

4. **稳定优先**
   选择社区成熟度高、维护稳定的技术组件。

5. **可扩展性**
   在保持简单的前提下，技术架构应具备一定扩展能力。

# 六、Embedding模型

Embedding 用于将文本转换为向量，以支持语义检索。

推荐模型：

```
BAAI/bge-small-zh
```

原因：

* 中文检索效果优秀
* 向量维度小（384）
* 推理速度快
* 非常适合 RAG

示例：

```python
from sentence_transformers import SentenceTransformer

model = SentenceTransformer("BAAI/bge-small-zh")

embedding = model.encode("如何申请退款")
```

# 七、Rerank模型

Rerank用于 **对检索结果进行二次排序**。

RAG流程：

```
Query
 ↓
Vector Search
 ↓
TopK Documents
 ↓
Rerank
 ↓
Top Context
 ↓
LLM
```

推荐模型：

```
BAAI/bge-reranker-base
```

作用：

```
(query, document) → relevance score
```

示例代码：

```python
from sentence_transformers import CrossEncoder

reranker = CrossEncoder("BAAI/bge-reranker-base")

pairs = [
 ("如何申请退款","用户可以在订单页面申请退款"),
 ("如何申请退款","API Key 创建方式")
]

scores = reranker.predict(pairs)
```
