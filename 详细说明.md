# SmartSupport AI

## 企业级智能客服 Agent（RAG + Memory + Tool Calling）

---

# 一、项目简介

SmartSupport AI 是一个基于 **Agent架构 + RAG（Retrieval-Augmented Generation）** 的企业智能客服系统。

系统支持：

* 企业知识库问答
* 用户上下文记忆
* Tool调用（订单 / 工单）
* Agent决策
* Chunk策略管理
* 检索策略管理
* Rerank检索优化

该项目用于展示 **AI Agent工程能力**。

---

# 二、业务背景

SaaS企业客服需要处理大量问题：

* 产品使用
* API接入
* 技术文档
* 错误排查
* 订单查询
* 工单查询

传统流程：

```
用户提问
↓
客服查文档
↓
客服查后台
↓
回复
```

效率低。

企业希望构建：

**AI客服Agent**

能力包括：

1 RAG知识库
2 用户记忆
3 Tool调用
4 Agent推理

### 2 Chunk策略管理

企业文档需要分块：

```

```

* title
* section

```

↓

↓
Vector Search
↓
Keyword Search
↓
```

### 4 Agent决策

Agent判断问题类型：

```
if query contains "订单":
    call order_tool

elif query contains "工单":
    call ticket_tool

else:
    use RAG
```

### 模块功能

#### 2.1 仪表盘 (Dashboard)

- 今日对话数量
- 活跃用户数
- 知识库文档数量
- 最近对话列表（脱敏）
- 查询类型分布（订单/工单/知识库）

#### 2.2 知识库管理

**文档列表**
- 表格展示所有文档
- 列：文件名、类型、大小、上传时间、状态、操作
- 支持搜索、筛选、分页
- 操作：查看、删除、重新索引

**文档上传**
- 拖拽上传区域
- 支持格式：.txt, .md, .pdf, .docx
- 上传进度显示
- 批量上传支持
- 上传后自动触发向量化

**文档预览**
- 点击文档名预览内容
- 显示分块结果
- 显示向量化状态

#### 2.3 策略配置

**Chunk策略配置**
```yaml
chunk_strategy:
  chunk_size: 500        # 滑动条 + 数字输入
  chunk_overlap: 100     # 滑动条 + 数字输入
  separators:            # 多行文本输入
    - "\n\n"
    - "\n"
    - "。"
```

**检索策略配置**
```yaml
retrieval_strategy:
  search_type: "hybrid"  # 下拉选择：vector/keyword/hybrid
  vector_top_k: 10       # 数字输入
  keyword_top_k: 10      # 数字输入
  rerank_top_k: 3        # 数字输入
  score_threshold: 0.5   # 滑动条 0-1
```

**记忆策略配置**
```yaml
short_term_memory:
  max_history_length: 5  # 数字输入
  enable_summary: false  # 开关

long_term_memory:
  enabled: true          # 开关
  max_topics_per_user: 10
  topic_expiry_days: 30
```

#### 2.4 系统监控

**对话统计**
- 时间范围选择（今日/本周/本月/自定义）
- 对话数量趋势图
- 查询类型饼图
- 响应时间分布
- Top 10 热门问题

**日志查看**
- 实时日志流
- 日志级别筛选
- 关键词搜索
- 日志导出
